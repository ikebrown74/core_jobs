#CORE Jobs

A job board built using Django and [djobberbase](https://github.com/wtrevino/django-djobberbase) for [Columbia Organization of Rising Entrepreneurs](http://www.columbia.edu/cu/core/)

##Requirements:

* Python 2.7+
* Django 1.3+
* `markdown`

##Frameworks Used:

* Foundation
* Bootstrap
* django-taggit
